Why is RCHGetElementNumber() returning "Undefined"?



RCHGetElementNumber funtion gets the Element Number definitions from files such as smf-elements-1.txt See Elements for details


  • If the element number you request doesn't have a definition in one of the elements files then you will get Undefined returned.

  • If you get Obsolete then there is a definition for the element but it has been marked Obsolete as the data is no longer available.

  • If you get Error then there is a definition for the element but, most likely, the data is no longer available, or the page has changed.
