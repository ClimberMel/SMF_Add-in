Documentation for smfWord() function

Description

A utility function to extract nth word from a string using a delimiter.

Syntax

=smfWord(Haystack, Occurrence, [Delimiter], [Convert])

Parameters

Haystack = String to extract the word from.
Occurance = The nth word found surrounded by the delimiter.
Delimiter = Optional - Default is “ “. String to delimit the Haystack into words. If a zero-length string is used; i.e., “”, Haystack will be returned.
Convert = Optional - Default is “0”. If set to 1, function will attempt to convert the substring to a numeric value, for example convert “50K” to “50000”.

Examples

  • To extract the 3rd word from "apples - oranges - bananas - grapes", you would use: =smfWord("apples - oranges - bananas - grapes") ... which gives bananas including the surrounding spaces.

  • To extract the 2nd word from "apples - oranges - bananas - grapes", you would use: =smfWord("apples - oranges - bananas - grapes", "2", " - ") ... which gives “oranges“; the delimiter includes spaces.

  • To extract the 3rd word from "Accounts Receivables: 700K", you would use: =smfWord("Accounts Receivables: 700K", "3", , 1) ... which gives “700000“ using the Convert parameter.
