Stock Market Functions Add-In Templates

As time goes on, I will be adding some templates and examples to the "Templates" folder of the Yahoo! Group's library. The main purpose of these templates will be to show how the add-in might be used. They will also provide some ranges of data that might easily be cut and pasted into a custom spreadsheet you wish to create.

Template Conventions
    
There are several standard conventions I'll try to use in the templates:
  1. Input fields (e.g. a stocks Ticker symbol) will have a background of yellow.

  2. If the entire template is based on a single Ticker symbol, the cell containing the Ticker symbol should be named "Ticker". This makes it easy to copy formulas from one workbook to another. When you copy a formula using a named range to another workbook and the new workbook contains the same named range, EXCEL will ask you if you want to change the named range reference to the field in the new workbook.

  3. Most RCHGetElementNumber() functions will use a hard-coded element number. This should also make it easier to copy cells and/or ranges from one workbook to another.
Update Links errors
    
Because EXCEL saves a hard-coded link to the location of an add-in's source code, opening these templates on another machine may cause a error with those links if you don't have the add-in stored in the recommended location (C:\Program Files\SMF Add-In).
